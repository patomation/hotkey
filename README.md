Hotkey
==========

A small module that handles your hotkeys

## Installation

  `npm install @patomation/hotkey`

## Usage
```javascript
hotkey.bind('control+z', () => {
    //Do some undo action
});
```

```javascript
hotkey.bind('enter', () => {
    //Do something when user hits enter
});
```

```javascript
hotkey.bind('downarrow', () => {
    //Down arrow function
});
```
